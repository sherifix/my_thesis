
# Master Thesis

This repo contain files for master thesis. The pipeline used here is available at [Cel_screening](https://github.com/sherifix/cel_screening)

## Overview

Implementation of [cellulase sreeening pipeline](https://github.com/sherifix/cel_screening) for master thesis. 

### System requirements 

- Linux OS (tested on Ubuntu)
- Minimum 16 GB RAM
- 30 GB free disk space
- Conda (Miniconda or Anaconda)
- Git
- Internet connection
- Available GUI for visualizations


### Cloning pipeline's repository

- Go to [cel_screening](https://github.com/sherifix/cel_screening): **There is comprehensive guidance for implementation of the following steps**
- Clone the repository 
- Create required conda environment
- Install external tools 

### Workflow to prepare input data 

The workflow used in the thesis is shown in [my_workflow.ipynb](https://github.com/sherifix/my_thesis/blob/master/my_workflow.ipynb)

### Source code for creating tables and figures

The code used for configuration of tables and figures used in the thesis is available in [figures_tables_generator.ipynb](https://github.com/sherifix/my_thesis/blob/master/figures_tables_generator.ipynb)
